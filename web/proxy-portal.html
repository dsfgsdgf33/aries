Use cURL with HTTP proxy:

Python requests:

import requests proxies = { "http": "http://USER:PASS@proxy.example.com:19901", "https": "http://USER:PASS@proxy.example.com:19901" } r = requests.get("https://httpbin.org/ip", proxies=proxies) print(r.json())

Browser proxy settings:

Proxy Type: HTTP Host: proxy.example.com Port: 19901 Username: USER Password: PASS Or use a browser extension like FoxyProxy / SwitchyOmega

# cURL with SOCKS5 curl --socks5 USER:PASS@proxy.example.com:19901 https://httpbin.org/ip # Python with PySocks import socks, socket socks.set_default_proxy(socks.SOCKS5, "proxy.example.com", 19901, username="USER", password="PASS") socket.socket = socks.socksocket
