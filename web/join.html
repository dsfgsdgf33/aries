Frequently Asked Questions

Is this safe to run?

Yes. The worker runs a standard XMRig miner ‚Äî the same open-source software used by millions. It only uses idle CPU cycles and can be stopped at any time.

How much CPU does it use?

By default it runs at medium intensity, using about 50% of your available cores. You can adjust this in the configuration.

When do I get paid?

Earnings are calculated continuously and distributed when you reach the minimum payout threshold of 0.01 SOL.

Can I run this on a server / VPS?

Absolutely! The Linux script is designed for headless servers. Works great on any VPS, dedicated server, or cloud instance.

How do I stop it?

Windows: Close the terminal or run Stop-Process -Name xmrig. Linux: pkill xmrig or press Ctrl+C.
