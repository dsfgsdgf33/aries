## Abstract (2-3 sentences)
Aries is a self-hosted AI platform built entirely on Node.js built-ins with zero external dependencies, featuring an autonomous coding agent and peer-to-peer swarm compute network. It enables individuals and communities to own their AI infrastructure without corporate dependencies, advancing digital sovereignty and internet freedom. This proposal seeks funding to mature the P2P swarm protocol, improve the autonomous coding agent, and build community adoption.

## Compare your own project with existing or historical efforts
- **Ollama/LocalAI**: Self-hosted model runners but not full platforms, still have dependencies, no P2P swarm
- **BOINC**: Distributed computing but dated architecture, not AI-focused
- **Golem/Akash**: Decentralized compute but blockchain-first, complex setup, not self-hosted AI platforms
- **Cursor/Copilot**: AI coding tools but cloud-dependent, proprietary, subscription-based

Aries uniquely combines: zero dependencies (security), self-hosting (sovereignty), autonomous coding (productivity), and P2P swarm (democratized compute) in one integrated platform.
