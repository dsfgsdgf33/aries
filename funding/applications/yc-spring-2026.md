## Describe what your company does in 50 characters or less
Self-hosted AI platform with P2P swarm compute

## What is your company going to make?
Aries is a self-hosted AI platform with zero npm dependencies, an autonomous coding agent that builds entire applications, and a P2P swarm compute network. Users own their AI — no subscriptions, no data sharing, no cloud dependency. Low-end machines contribute via crypto mining, creating a community compute economy where everyone participates.

## Why did you pick this idea to work on?
I've been using AI tools daily and every one requires a subscription, sends my code to someone's cloud, and could be shut down tomorrow. I wanted AI I actually own. When I started building it, I realized the entire Node.js ecosystem is built on layers of dependencies nobody audits. So I built Aries from scratch — zero dependencies, just Node.js built-ins. It took longer but the result is something genuinely new: a secure, self-contained AI platform anyone can run.

## What's new about what you're making?
1. Zero npm dependencies — unique in the Node.js ecosystem at this scale. No supply chain risk.
2. Autonomous coding agent that plans, builds, tests, and fixes complete apps — not just autocomplete.
3. P2P swarm compute with mining fallback — every machine contributes, creating an inclusive AI economy.

## How far along are you?
Working product. 42+ integrated tools, autonomous coding agent operational, cyberpunk web dashboard, multi-provider AI support (OpenAI, Anthropic, local models). P2P swarm architecture designed, implementation in progress.

## How long have each of you been working on this?
~1 year, full-time effort.

## How do you know people need what you're making?
The self-hosted AI movement is exploding — Ollama has 100K+ GitHub stars, LocalAI has 25K+. But none are complete platforms, and all have hundreds of dependencies. There's massive demand for AI tools that respect privacy and ownership. The zero-dependency angle is a security story enterprises care about deeply.

4. Premium features (advanced autonomous coding workflows, priority compute)
