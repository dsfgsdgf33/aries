# Mozilla MOSS Application Draft — Aries (Track II: Mission Partners)

**Submit at**: https://www.mozilla.org/en-US/moss/

## Project Name
Aries — Self-Hosted AI Platform

## How does Aries advance Mozilla's mission?
Mozilla's mission is an internet that is a global public resource, open and accessible to all. Aries directly advances this by:

1. **Decentralizing AI**: AI is becoming the interface to the internet. Aries ensures this interface isn't controlled by corporations.
2. **Privacy by default**: Self-hosted means user data never leaves their machine. No telemetry, no tracking, no cloud dependency.
3. **Open source security**: Zero npm dependencies means zero supply chain attack surface — a genuine security innovation for the Node.js ecosystem.
4. **Community-owned compute**: The P2P swarm democratizes AI infrastructure, making it accessible regardless of economic status.

## Project Description
Aries is a fully self-hosted AI platform built on Node.js with zero external dependencies. It includes an autonomous coding agent, 42+ integrated tools, multi-provider AI support, and a P2P swarm compute network. MIT licensed.

## Requested Amount
$50,000

## What will the funds be used for?
- P2P swarm compute network development
- Security hardening of zero-dependency codebase
- User documentation and accessibility improvements
- Community building and contributor onboarding
