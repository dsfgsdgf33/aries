# Ethereum Foundation ESP Application Draft — Aries

**Submit at**: https://esp.ethereum.foundation/

## Project Name
Aries Swarm: Decentralized AI Compute Infrastructure

## Project Description
Aries is building a P2P swarm compute network for distributed AI inference and training, with a crypto mining fallback that creates an inclusive compute economy. The platform is fully open source (MIT), self-hosted, and built with zero external dependencies on Node.js built-ins only.

We're seeking ESP support to develop the decentralized compute layer that could integrate with or complement Ethereum's ecosystem — enabling verifiable AI computation, decentralized model serving, and community-owned AI infrastructure.

## How does this benefit the Ethereum ecosystem?
1. **Decentralized AI compute**: Provides infrastructure for AI inference that doesn't depend on centralized cloud providers — aligned with Ethereum's decentralization ethos
2. **Public goods**: MIT-licensed, zero-dependency platform that anyone can run and verify
3. **Compute verification**: Research into verifiable computation for AI tasks in trustless environments
4. **Builder tools**: Autonomous coding agent (Aries Code) that could be specialized for Solidity/smart contract development
