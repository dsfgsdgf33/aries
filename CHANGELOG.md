# Changelog

## v5.3.2 — One-Click Installers (2025-02-18)
- **Windows one-click installer** (`install-windows.bat`) — auto-installs Node.js, dependencies, runs setup
- **Linux/macOS installer** (`install.sh`) — auto-detects distro, installs Node.js via apt/dnf/pacman/brew
- **Remote one-liner installers** — `irm .../install.ps1 | iex` (Windows) / `curl ... | bash` (Linux/Mac)
- **Desktop shortcuts** — auto-created on Windows (desktop + Start Menu), Linux (.desktop), macOS (shell alias)
- **First-run detection** — launcher.js auto-runs setup if config.json is missing
- Updated README with multiple installation options
