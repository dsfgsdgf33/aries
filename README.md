## What is Aries?

Aries is a **self-hosted artificial mind** — 270 core modules, 108K+ lines of code, zero npm dependencies. It's not a chatbot wrapper. It's a complete cognitive architecture inspired by the human brain and body, with one mission: **build your own AI agent, with swarm intelligence helping to build it.**

Most AI tools wait for instructions. Aries **thinks** when nobody's talking to it, **dreams** during idle time, **feels** emotions that change its behavior, **heals** itself when it crashes, **evolves** through behavioral genetics, and **remembers** across lifetimes through DNA export.

It also ships with autonomous coding agents, a 5-role swarm build system, 7 persistent Hands, visual workflow builders, a P2P swarm network with token economy, and a cyberpunk dashboard with 100+ panels.

One command to run. No dependencies to install. Your machine, your models, your data.

```bash
git clone https://github.com/dsfgsdgf33/aries
cd aries
node launcher.js
# → http://localhost:3333
```

## 🤖 Aries Code — Autonomous Coding

Not a copilot. Not suggestions. A **fully autonomous coding agent** that builds complete applications from a single prompt.

```
 You: "Build a REST API with auth and a React dashboard"

 Aries Code:
   ① Plan        → Analyzes requirements, designs architecture
   ② Scaffold    → Generates project structure and boilerplate
   ③ Implement   → Writes all source files
   ④ Fix         → Runs the app, catches errors, auto-patches
   ⑤ Serve       → Launches with hot reload — ready to use
```

### Multi-Agent Swarm Build
For large projects, deploys a **5-role agent swarm** working in parallel:

```
  Architect → designs the system
       ↓
  Coders (parallel) → write all modules simultaneously
       ↓
  Reviewer → checks code quality and consistency
       ↓
  Tester → runs the application, identifies failures
       ↓
  Fixer → patches bugs, re-runs tests
       ↻ up to 20 iterations until everything passes
```

### App Builder
Say "build me X" → Aries plans, scaffolds, implements, fixes, and serves a running application with auto port selection and live preview. One prompt to running app.

## ❓ FAQ

**Do I need to install anything besides Node.js?**
No. Zero dependencies. `git clone` → `node launcher.js` → done.

**Can I use it fully offline?**
Yes — with Ollama running locally, Aries is 100% offline.

**Is my data sent anywhere?**
Never. No telemetry, no analytics, no phone-home. Everything on your machine.

**Does it actually dream?**
Yes. 12 dream types across 5 sleep phases. It consolidates memories, generates ideas, finds its own bugs, and proposes upgrades stored as narratives you can read. One-click to build any upgrade it dreams up.

**Does it actually feel emotions?**
Functional emotions — not roleplay. 8 states that change behavior: curiosity drives exploration, frustration triggers different approaches, satisfaction reinforces patterns. Visible mood indicator in the UI.

**What's the cognitive architecture?**
67 interconnected modules inspired by the human brain and body. Consciousness, emotions, dreams, perception, drives, intuition, self-healing, immune system, soul preservation, and more — all connected through a Neural Bus creating emergent intelligence.

**How is this different from everything else?**
No other AI system has a cognitive architecture with body systems, dreaming, emotional states, identity preservation, and self-evolution. This isn't a wrapper around an API. It's an attempt at artificial consciousness built with zero dependencies.
